Simon Says Game!

How to Play

  • Simon will give the first signal. One color tile will light up, accompanined by a tone.
  • Repeat the signal by touching the selecting the same color tile.
  • Simon will repeat the first signal and then add one more.
  • Repeat those two signals in the same order.
  • Simon will continue to play the sequence and add one.
  • Continue to repeat Simon for as long as you can.
  • If you fail to complete the sequence in the correct order the game will end.
